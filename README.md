# SetFlow

Setlist builder for musicians and production managers.

## Features

- **Song Library** - Store songs with BPM, key, energy level, duration
- **Setlist Builder** - Drag-and-drop song ordering
- **Flow Warnings** - Alerts for key clashes, energy drops, slow sequences
- **Time Tracking** - Target set length with changeover buffer
- **PDF Export** - Print-ready setlists for stage/FOH

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

React 18 • Vite 5 • Tailwind CSS • shadcn/ui • Framer Motion

## Deployment

Deploy to Vercel with zero config - just connect the repo.
